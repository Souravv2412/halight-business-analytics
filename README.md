# Halight Business Analytics

## Overview
**Halight Business Analytics** is a real-time TV dashboard and business intelligence platform designed to monitor and analyze user engagement for Halight. The platform provides actionable insights into user behavior, gamification performance, verification status, referrals, and retention patterns. This repository contains the full project including dashboard code, exploratory analysis, and documentation.

## Features

- **Real-time Data Connection**: Connects directly to Halight’s Athena database for up-to-date analytics.
- **User Engagement Analytics**: Tracks sign-ups, logins, learning module completions, gamification achievements, and point balances.
- **Verification & Referral Tracking**: Measures verification completion rates and referral program effectiveness.
- **Geographical Insights**: Analysis of user distribution across countries and regions.
- **Retention & Reactivation Insights**: Identifies inactive or disabled users for targeted reactivation campaigns.
- **Reward System Analysis**: Evaluates high-value user segments and retailer performance gaps.
